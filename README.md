📊 P&L Dashboard (Profit & Loss Analysis)
🔍 Overview

This project presents a comprehensive Profit & Loss (P&L) Dashboard built using Power BI to analyze financial performance across revenue, cost, and profitability dimensions.

The dashboard helps identify loss drivers, cost inefficiencies, and performance gaps using interactive visualizations and data-driven insights.

🟢 1. Overview
Revenue vs Budget (Gauge Chart)

👉 Insights:

Revenue is close to budget but not sufficient to cover total costs.

🟡 2. Analysis
Cost Breakdown (Donut Chart)
Loss by Region (Bar Chart)
Budget vs Actual Performance (Monthly Trend)
Interactive slicers (Year, Region, Product)

👉 Insights:

Majority of cost is driven by core operational expenses (~69%).
West and South regions contribute highest losses.
Actual revenue consistently underperforms compared to budget.
Declining trend after mid-year indicates operational inefficiencies.

🔵 3. Key Insights

The business is running at a net loss, with expenses exceeding revenue.
Profit margins are negative across multiple segments.
High operational costs are the primary driver of losses.
Underperformance against budget highlights weak sales execution.
Regional inefficiencies significantly impact overall profitability.
Loss trend is consistent over time, not a one-time issue.
